📝 문제 설명: 연속된 부분 수열의 합
비내림차순으로 정렬된 수열이 주어질 때, 다음 조건을 만족하는 부분 수열을 찾으려고 합니다.

조건

기존 수열에서 임의의 두 인덱스의 원소와 그 사이의 원소를 모두 포함하는 연속된 부분 수열이어야 합니다.

부분 수열의 합은 **k**입니다.

합이 k인 부분 수열이 여러 개인 경우, 길이가 가장 짧은 수열을 찾습니다.

길이가 짧은 수열이 여러 개인 경우, **앞쪽(시작 인덱스가 작은)**에 나오는 수열을 찾습니다.

제한사항

5≤ sequence의 길이 ≤1,000,000

1≤ sequence의 원소 ≤1,000

sequence는 비내림차순으로 정렬되어 있습니다.

5≤k≤1,000,000,000

k는 항상 sequence의 부분 수열로 만들 수 있는 값임이 보장됩니다.

입출력 예

sequence	k	result
[1, 2, 3, 4, 5]	7	[2, 3]
[1, 1, 1, 2, 3, 4, 5]	5	[6, 6]
[2, 2, 2, 2, 2]	6	[0, 2]
입출력 예 설명

예제 #1 [1, 2, 3, 4, 5]에서 합이 7인 연속된 부분 수열은 [3, 4]뿐입니다. 따라서 시작 인덱스 2와 마지막 인덱스 3을 담은 [2, 3]을 반환합니다.

예제 #2 [1, 1, 1, 2, 3, 4, 5]에서 합이 5인 연속된 부분 수열은 다음과 같습니다.

[1, 1, 1, 2] (인덱스 0~3, 길이 4)

[2, 3] (인덱스 3~4, 길이 2)

[5] (인덱스 6~6, 길이 1) 이 중 가장 길이가 짧은 것은 [5]이므로 [6, 6]을 반환합니다.

예제 #3 [2, 2, 2, 2, 2]에서 합이 6인 연속된 부분 수열은 [2, 2, 2]로 총 3가지 경우가 있습니다.

인덱스 0~2

인덱스 1~3

인덱스 2~4 길이가 모두 3으로 같으므로, 가장 앞선 인덱스인 [0, 2]를 반환합니다.
